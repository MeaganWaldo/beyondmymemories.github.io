This folder contains CSS files for our project. 

index.css: This CSS file is used to define the design and locations of specific items within our html files. Specifically, this handles the website entry scene and the character selection. 

chatLog.css: The CSS file that controls the look and layout of the chat log.

container1.css: The CSS file that controls the look and layout of the title screen.

container2.css: The CSS file that controls the look and layout of the character select.

container3.css: The CSS file that controls the look and layout of the main game page.

container4.css: The CSS file that controls the look and layout of the main combat page.

DiceStyleSheet.css: This CSS file controls the look and layout of the dice display.
