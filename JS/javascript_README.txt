This folder contains the JavaScript files for our project.

save.js: Contains JavaScript functions and code that will be utilized in our project to allow the player to save their progress in the game. 

idle.js: Contains JavaScript functions and code that will be utilized in our project to have the idle mechanic function. 

chatLog.js: Contains JavaScript functions and code that will be utilized in our project to have the chat log work.

designScripts.js: Contains JavaScript functions and code that will be utilized in our project for a hover effect.

diceFunction.js: Contains JavaScript functions and code that will be utilized in our project for dice rolling.

load_fuction.js: Contains JavaScript functions and code that will load the saved data back into the game.
